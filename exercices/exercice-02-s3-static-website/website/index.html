Qu'est-ce qu'AWS ?

Amazon Web Services est la plateforme cloud la plus complète et largement adoptée au monde.

Paiement à l'utilisation

Payez uniquement pour les ressources que vous consommez, sans engagement long terme.

Scalabilité instantanée

Augmentez ou réduisez vos ressources en quelques clics selon vos besoins.

Présence mondiale

Déployez vos applications au plus près de vos utilisateurs dans le monde entier.

Stockage

Solutions de stockage durables et évolutives

Amazon S3

Stockage objet hautement scalable. Idéal pour les backups, sites statiques, et data lakes.

Classes de stockage :

  • • Standard - Accès fréquent
  • • Infrequent Access - Accès occasionnel
  • • Glacier - Archivage long terme
Free Tier 5 Go

Amazon EBS

Volumes de stockage bloc persistants pour vos instances EC2. Haute performance.

Types de volumes :

  • • gp3 - Usage général SSD
  • • io2 - IOPS provisionnés
  • • st1 - HDD optimisé débit
Free Tier 30 Go

Amazon EFS

Système de fichiers élastique et partagé. Accessible depuis plusieurs instances EC2.

Avantages :

  • • Scalabilité automatique
  • • Multi-AZ par défaut
  • • Compatible POSIX
